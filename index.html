Application for Media organisation

The goal of this project is to create a web application that helps company teams organize, track, and manage their work more efficiently.

Key features:

  • Data Centralisation: All MOM's, Briefs, Jobs, LMS, Ideas are in one place to access anytime whenever needed.
  • Team collaboration: Users should be able to collaborate with team members by commenting on tasks, sharing files, and sending messages.
  • Reporting: Users should be able to generate reports on tasks, projects, and team performance. Reports should be customizable and exportable to different formats.
  • PWA: PWA with notifications with firebase integrations.
  • Customization: Users can be customize the application's dashboard and listing appearance.

The application is built using laravel and MySQL for the back-end, with HTML, CSS, and JavaScript for the front-end. The application is responsive, accessible, and optimized for performance and security.

Application for KPI/KRA

The Application for KPIs and KRAs is a tool designed to streamline performance management and enhance organizational effectiveness.

Key features:

  • Goal Setting: The application provides a structured framework for defining SMART goals, ensuring clarity and alignment with organizational objectives. Users can set specific, measurable, achievable, relevant, and time-bound KPIs and KRAs.
  • Progress Tracking: Users can easily input and update their performance data, allowing for real-time tracking of individual or team progress to provide insights into performance trends and areas that require attention.
  • Performance Evaluation: The application supports performance evaluations based on predefined KPIs and KRAs. This helps a transparent and objective performance review process.
  • Collaboration and Communication: The application facilitates collaboration among team members. Team member/ clients can share their views and insights, promoting accountability and teamwork. Managers can view collective performance, enabling them to identify strengths and areas for optimization.

The Application for KPIs and KRAs provides organizations with a robust platform to effectively manage performance. With features such as goal setting, progress tracking, performance evaluation, collaboration.
